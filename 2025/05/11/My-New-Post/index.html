基于hexo + Github 搭建一个专属自己的个人博客（已有github及账号版）

一、first step​​安装Node.js​​：Hexo基于Node.js（https://nodejs.org/zh-cn）

在 ​​普通命令行（CMD/PowerShell，快捷键win+r打开，CMD）​​ 中运行（不要在 Node.js REPL 环境输入）
输入：npm install -g hexo-cli

安装完成后验证：
hexo -v # 查看 Hexo 版本

二、​​初始化博客项目​​

hexo init myblog # 创建名为 myblog 的博客文件夹
cd myblog # 进入项目目录
npm install # 安装依赖包

三、本地启动测试​​

hexo server
访问 http://localhost:4000 查看默认博客页面。

一、搭建过程中常见的命令

cd myblog && dir

1.cd myblog:change directory
2.&&:”前一个命令成功执行后才执行下一个命令”
3.dir:显示当前目录内容的命令（类似 Linux 的 ls）

myblog && echo 路径正确 || echo 路径错误

echo是用于 输出文本的命令
1.&&（逻辑与）​
如果 myblog 执行成功（返回 0）​​，则执行 echo 路径正确。
2.||（逻辑或）​
如果 myblog 执行失败（返回非 0）​​，则执行 echo 路径错误。

hexo clean && hexo server

hexo clean # 1. 清理旧缓存

&& # 2. 如果上一步成功，则继续

hexo server # 3. 启动本地服务器

其他常见问题

如何重新打开博客编辑修改

在命令行里找到自己的博客目录
C:\Users\[自己用户名]\myblog# Windows系统

dir验证路径,应看到以下关键目录：
_config.yml source/ themes/ node_modules/

进入文章目录，用编辑器打开文章

cd source/_posts

code My-New-Post.md
