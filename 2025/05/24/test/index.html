API调用入门

什么是api，什么是json

1.API（Application Programming Interface）的通俗解释：

API 就像餐厅里的​​服务员​​，连接你和厨房。
当你想点餐时，不需要直接进厨房告诉厨师怎么做，只需告诉服务员（API）你的需求，它会帮你传递信息并端回食物。

2.JSON（JavaScript对象表示法）的简单说明：
JSON 是一种​​轻量级的数据格式​​，像快递盒的标签，清晰标注内容。

​​结构​​：用 {} 包裹“键值对”，例如：
{ "name": "小明", "age": 25, "hobbies": ["篮球", "读书"] }
在软件中，API 是不同程序之间沟通的桥梁。例如：微信登录其他App时，调用了微信的API来验证你的身份。

3.API和JSON的关系：

​​数据传递​​：API 常通过 JSON 格式发送和接收数据。
***

import json的用途

  • 生成json

import json

json_string = ‘{“name”: “小明”, “age”: 25}’ # 模拟API返回的JSON字符串

data = json.loads(json_string) # 解析为Python字典

print(data[“name”]) # 输出：小明

  • 解析json
    import json

user_data = {“name”: “小红”, “age”: 22}

json_string = json.dumps(user_data) # 转换为JSON格式的字符串

print(json_string) # 输出：{“name”:”\u5c0f\u7ea2”, “age”: 22}

  • json.loads()：将 ​​JSON字符串​​ 解析为 ​​Python字典/列表​​。
  • json.dumps()：将 ​​Python字典/列表​​ 转换为 ​​JSON字符串​​。
  • json.load()：从 ​​JSON文件​​ 读取数据并解析为 Python 对象。
  • json.dump()：将 Python 对象写入 ​​JSON文件​​
