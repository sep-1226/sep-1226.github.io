My New Post

еҹәдәҺhexo + Github жҗӯе»әдёҖдёӘдё“еұһиҮӘе·ұзҡ„дёӘдәәеҚҡе®ўпјҲе·ІжңүgithubеҸҠиҙҰеҸ·зүҲпјү

дёҖгҖҒfirst stepвҖӢвҖӢе®үиЈ…Node.jsвҖӢвҖӢпјҡHexoеҹәдәҺNode.jsпјҲhttps://nodejs.org/zh-cnпјү

еңЁ вҖӢвҖӢжҷ®йҖҡе‘Ҫд»ӨиЎҢпјҲCMD/PowerShellпјҢеҝ«жҚ·й”®win+rжү“ејҖпјҢCMDпјүвҖӢвҖӢ дёӯиҝҗиЎҢпјҲдёҚиҰҒеңЁ Node.js REPL зҺҜеўғиҫ“е…Ҙпјү
иҫ“е…Ҙпјҡnpm install -g hexo-cli

е®үиЈ…е®ҢжҲҗеҗҺйӘҢиҜҒпјҡ
hexo -v # жҹҘзңӢ Hexo зүҲжң¬

дәҢгҖҒвҖӢвҖӢеҲқе§ӢеҢ–еҚҡе®ўйЎ№зӣ®вҖӢвҖӢ

hexo init myblog # еҲӣе»әеҗҚдёә myblog зҡ„еҚҡе®ўж–Үд»¶еӨ№
cd myblog # иҝӣе…ҘйЎ№зӣ®зӣ®еҪ•
npm install # е®үиЈ…дҫқиө–еҢ…

дёүгҖҒжң¬ең°еҗҜеҠЁжөӢиҜ•вҖӢвҖӢ

hexo server
и®ҝй—® http://localhost:4000 жҹҘзңӢй»ҳи®ӨеҚҡе®ўйЎөйқўгҖӮ

дёҖгҖҒжҗӯе»әиҝҮзЁӢдёӯеёёи§Ғзҡ„е‘Ҫд»Ө

cd myblog && dir

1.cd myblog:change directory
2.&&:вҖқеүҚдёҖдёӘе‘Ҫд»ӨжҲҗеҠҹжү§иЎҢеҗҺжүҚжү§иЎҢдёӢдёҖдёӘе‘Ҫд»ӨвҖқ
3.dir:жҳҫзӨәеҪ“еүҚзӣ®еҪ•еҶ…е®№зҡ„е‘Ҫд»ӨпјҲзұ»дјј Linux зҡ„ lsпјү

myblog && echo и·Ҝеҫ„жӯЈзЎ® || echo и·Ҝеҫ„й”ҷиҜҜ

echoжҳҜз”ЁдәҺ иҫ“еҮәж–Үжң¬зҡ„е‘Ҫд»Ө
1.&&пјҲйҖ»иҫ‘дёҺпјүвҖӢ
еҰӮжһң myblog жү§иЎҢжҲҗеҠҹпјҲиҝ”еӣһ 0пјүвҖӢвҖӢпјҢеҲҷжү§иЎҢ echo и·Ҝеҫ„жӯЈзЎ®гҖӮ
2.||пјҲйҖ»иҫ‘жҲ–пјүвҖӢ
еҰӮжһң myblog жү§иЎҢеӨұиҙҘпјҲиҝ”еӣһйқһ 0пјүвҖӢвҖӢпјҢеҲҷжү§иЎҢ echo и·Ҝеҫ„й”ҷиҜҜгҖӮ

hexo clean && hexo server

hexo clean # 1. жё…зҗҶж—§зј“еӯҳ

&& # 2. еҰӮжһңдёҠдёҖжӯҘжҲҗеҠҹпјҢеҲҷз»§з»ӯ

hexo server # 3. еҗҜеҠЁжң¬ең°жңҚеҠЎеҷЁ

е…¶д»–еёёи§Ғй—®йўҳ

еҰӮдҪ•йҮҚж–°жү“ејҖеҚҡе®ўзј–иҫ‘дҝ®ж”№

еңЁе‘Ҫд»ӨиЎҢйҮҢжүҫеҲ°иҮӘе·ұзҡ„еҚҡе®ўзӣ®еҪ•
C:\Users\[иҮӘе·ұз”ЁжҲ·еҗҚ]\myblog# Windowsзі»з»ҹ

dirйӘҢиҜҒи·Ҝеҫ„,еә”зңӢеҲ°д»ҘдёӢе…ій”®зӣ®еҪ•пјҡ
_config.yml source/ themes/ node_modules/

иҝӣе…Ҙж–Үз« зӣ®еҪ•пјҢз”Ёзј–иҫ‘еҷЁжү“ејҖж–Үз«
