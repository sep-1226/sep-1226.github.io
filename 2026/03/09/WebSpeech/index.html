基于 Web Speech API 的浏览器实时语音识别 Demo

项目简介

这是一个基于 浏览器原生 Web Speech API 实现的前端语音识别 Demo，主要目标是让页面具备 实时语音转文字 的能力，并尽量降低接入成本。

和传统的“录音 → 上传服务器 → 调用 ASR 接口 → 返回识别结果”的方案不同，这个项目直接利用浏览器提供的原生能力，在前端完成语音识别，适合用于：

  • AI 对话输入
  • 语音搜索
  • 表单语音填写
  • 轻量级语音助手
  • 快速原型验证

项目亮点

1. 实时识别，边说边出结果

用户说话时，页面可以同步显示识别文本，交互体验更自然，适合对话式场景。

2. 纯前端能力接入成本低

项目基于浏览器原生 Web Speech API，不依赖额外后端服务，不需要自己处理音频上传、转码和识别链路，适合快速开发。

3. 支持多语言识别

方案可支持中文、英文、日文、韩文等多种语言，在不同产品场景下具备较好的扩展性。

4. 适合封装成可复用组件

项目中已将语音识别能力抽象为组件，可以在不同页面中快速复用，降低业务接入成本。

技术栈

  • Next.js
  • TypeScript
  • Tailwind CSS
  • Web Speech API

功能说明

目前项目已经实现了以下能力：

  • 浏览器端语音识别
  • 实时回显识别结果
  • 识别完成后的回调处理
  • 多语言配置支持
  • 组件化封装，便于业务接入

项目中的核心用法类似这样：

import { SpeechRecognitionDemo } from '@/components/SpeechRecognitionDemo';

export default function Page() {
    const handleTranscriptComplete = (text: string) => {
        console.log('识别结果：', text);
        // 在这里处理识别完成后的文本
    };

    return (
        <div>
            <SpeechRecognitionDemo
                onTranscriptComplete={handleTranscriptComplete}
            />
        </div>
    );
}
