我做了一个用于开题报告写作的 Claude Skill：以“阿列西·艾尔雅维奇的图像意识形态研究”为例

Posted on 2026-03-08 In AI Projects
这是一个面向中文人文学术写作场景的 Claude Skill，能够围绕特定研究对象自动生成开题报告框架、研究现状梳理、方法论设计与学术表达优化。

项目背景

最近我在尝试把 AI 从“聊天工具”变成“可复用的专业能力模块”。
相比一次性的提问，我更关心的是：能不能把某一类复杂任务沉淀成稳定可复用的 Skill（技能），让模型在特定场景下持续输出更可靠、更结构化的结果。

这次我做的，就是一个偏人文学术场景的 Claude Skill：

开题报告 Skill：阿列西·艾尔雅维奇的图像意识形态研究

它面向的任务不是泛泛的“帮我写点内容”，而是更具体的学术写作需求，比如：

  • 围绕某位理论家的思想生成开题报告初稿
  • 自动拆分“选题意义 / 国内外研究现状 / 研究方法 / 创新点 / 计划进度”
  • 保持人文学术表达风格，而不是生成泛泛的 AI 套话
  • 尽量让结果接近真正可修改、可继续打磨的研究文本

我为什么要做这个 Skill

在实际写作中，我发现学术写作里的困难并不只是“不会写”，而是：

  1. 资料很多，但难以形成结构
  2. 知道主题，但不知道如何进入理论脉络
  3. 会表达观点，但不会写成标准学术格式
  4. AI 常常写得很空，缺乏研究问题意识
  5. 不同章节风格不统一，容易像拼接稿

尤其是在人文学科里，开题报告并不只是一个模板填空任务，它要求写作者能同时处理：

  • 研究对象
  • 学术史脉络
  • 核心概念
  • 研究方法
  • 理论价值与现实意义
  • 学术表达的严谨性

所以我想做的，不是“生成一篇看起来像论文的东西”，而是设计一个能够辅助研究构思的 Skill。

这个 Claude Skill 是做什么的

这个 Skill 主要负责以下几类任务：

1. 生成开题报告整体框架

当输入研究主题后，它能够先给出较为规范的开题结构，例如：

  • 摘要
  • 关键词
  • 选题目的与意义
  • 国内外研究现状
  • 研究内容
  • 研究方法
  • 重点难点
  • 创新点
  • 进度安排
  • 预期成果

这样做的好处是，写作一开始就不再是“从空白页开始”。

2. 帮助梳理理论脉络

以“阿列西·艾尔雅维奇”这一研究对象为例，Skill 会重点围绕以下问题展开组织：

  • 他的图像理论放在什么思想谱系中理解？
  • “图像”在其理论体系中承担怎样的功能？
  • 它如何与意识形态批判、审美判断、历史语境发生关系？
  • 与视觉文化研究、图像理论、马克思主义美学之间有什么关联？

这类输出比普通的“人物介绍式总结”更接近学术写作真正需要的内容。

3. 调整语言为学术表达

我在设计时尤其强调了一点：
输出不能只是“像说明文”，而要尽量接近中文人文学术写作的语体。

例如，它会尽量避免：

  • 过度口语化
  • 套话式“具有重要意义”
  • 空泛的宏大表述
  • 缺乏主谓逻辑的堆词

而更倾向于输出：

  • 结构清晰的论述句
  • 能进入具体理论问题的表述
  • 适合直接进入开题报告正文的学术语体

4. 作为“写作脚手架”而不是“一键代写”

我并不把这个 Skill 理解为代写工具。
更准确地说，它像一个学术写作脚手架：

  • 帮你搭结构
  • 帮你找进入问题的角度
  • 帮你统一表达风格
  • 帮你从“知道主题”走到“形成研究问题”

最后真正的论文，仍然需要研究者自己判断、修改、补充文献与论证。

Skill 的工作流设计

这个项目本质上不是一个单独 prompt，而是一个相对清晰的工作流。

第一步：输入研究主题与基本方向

例如：

  • 研究对象是谁
  • 研究聚焦什么问题
  • 想从哪个理论视角切入
  • 需要生成哪种文体（开题报告、综述、研究计划等）

第二步：先生成结构，而不是直接堆正文

这是我比较重视的一点。
很多 AI 写作失败的原因在于：一上来就让模型写长文，结果容易散、空、重复。

所以我会先让 Skill 输出：

  • 章节骨架
  • 每部分核心问题
  • 各部分之间的逻辑关系

第三步：逐段展开

然后再继续生成：

  • 摘要
  • 研究意义
  • 研究现状
  • 方法论
  • 创新点
  • 进度安排

这样生成出来的文本通常比一次性长文更稳定。

第四步：统一风格并进行学术化修订

最后再做一轮处理：

  • 压缩空话
  • 提高术语准确度
  • 强化逻辑连接
  • 调整为偏中文学术写作的句式

这次案例：阿列西·艾尔雅维奇的图像意识形态研究

这次 Skill 的测试案例，聚焦的是：

阿列西·艾尔雅维奇的图像意识形态研究

之所以选这个题目，是因为它非常适合检验一个学术写作 Skill 是否真的有用。
因为这个题目同时包含几个难点：

  • 研究对象相对小众，不像福柯、阿多诺那样语料现成
  • 涉及“图像”“意识形态”“视觉文化”等多个理论层面
  • 不能只写成思想介绍，必须形成研究问题
  • 需要在中文学术表达中保持严谨性和概念分辨率

从目前的测试结果来看，这个 Skill 在以下方面表现比较稳定：

  • 能较快形成开题整体框架
  • 能围绕“图像”组织理论问题，而不是只介绍人物生平
  • 能输出较为规整的开题语体
  • 能减少无意义套话，提高文本可修改性
